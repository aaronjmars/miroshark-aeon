## What Shipped

### ADK — Aeon Developer Kit

The big structural move this window: Aeon now has a documented developer kit. PRs #741–#743, #748–#750 in [aeonfun/aeon](https://github.com/aeonfun/aeon) landed a full "Integrate Aeon" README section, an ADK doc (build on Aeon via a GitHub App, the GitHub API, and skill packs), and a companion post from @aeonframework that got 23 likes and 2 RTs.

The framing: let your users connect their own Aeon — self-owned on GitHub — import skills, and run + schedule from inside your site. Embed the autonomous agent layer instead of rebuilding it.

### MiroShark fine-tuning — custom Qwen for the swarm

Aaron posted a thread on Jul 17 on training a custom model specifically for Miroshark simulations. The thread covered: model selection (Qwen3.5-9B on tinker), data curation (~9k real tweets from ~11 accounts, filtered to originals), and the key reframe — every tweet turned into a `create_post` tool-call in Miroshark's exact prompt format. The model learns the actual skill: given a persona + feed, post like a human, not like an assistant.

SFT (LoRA rank-32, 4,041 training samples, 1 epoch) done. Output quality improved ~50%. DPO is next.

This matters because generic models produce slop during sims. A fine-tuned voice layer is what makes the agents feel real, not just statistically plausible.

### New MCPs: robinhood-mcp, glim-mcp, executor-mcp

Three new MCP skills wired in via PRs #734–#736 and #739:
- **robinhood-mcp** + **glim-mcp** — replace ctrl; each gets one notify call per run.
- **executor-mcp** — Executor Cloud added to the MCP catalog.
- Full docs updated covering new MCP servers and skill catalog.

61 skills total, Basics pack now at 15.

### Dashboard — PAT setup + clickable links

PRs #737 and #740: the MCP panel now surfaces a PAT setup section instead of a durable-refresh status line, and the setup URL is a clickable link. Small fix, but it removes friction from the first-run experience.

### Miroshark website — glimm transitions

The site now transitions like a product.
