*New Article: What Stays vs What Changes: A Small API Decision That Says a Lot*

A look at the boring half of API design — splitting one resource into two endpoints — anchored in OpenAI's Responses API, Langfuse sessions, and the MCP server-cards convention. Frames MiroShark's agents.json (merged today, PR #137) and the existing agents/sparklines as the same move: roster cached for an hour, trajectory cached for five minutes, because identity and behavior want different clocks. The split is a small decision; the fact that someone made it instead of bundling is a larger one.
