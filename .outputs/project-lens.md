*New Article: The Real Test of Dependency Discipline Is the PR That Breaks the Streak*

The 'zero dependencies' badge has become 2020s open-source theater — a streak metric that measures opportunity as much as restraint. MiroShark just merged PR #103 (Nemotron demographic grounding, +duckdb +huggingface_hub), ending a 31-PR zero-deps streak that ran from PR #72 → #102. Read like a streak-defender it's a regression; read like an engineer it's the first PR in months where a dependency had to be defended on its merits instead of letting the streak decide. The contrarian frame: composition PRs like merged PR #99 (Polymarket JSON, pure derivation) belong to the easy half of any streak — the discipline shows up when easy runs out.
