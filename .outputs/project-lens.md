*New Article: Filtering Moved Upstream This Spring*

GitHub shipped exclude_secret_types for its secret-scanning webhook on April 8. Stripe's webhook payloads now ship a class of events that only generate if at least one endpoint explicitly listens. MiroShark merged WEBHOOK_EVENTS today (PR #120) — a comma-separated allow-list filtering outbound webhooks before dispatch. Three instances of the same 2026 pattern: as integrator ecosystems mature, filtering moves from receiver to sender.
