## What Shipped

### MiroShark (`aaronjmars/MiroShark`)

11 PRs merged, 11 commits. Mostly correctness and hardening — plus one big locale push.

**French locale (PR #222)** — contributor Zarbel974 landed full French frontend coverage: ~1,627 `tr()` calls, completing the FR translation pass. MiroShark was already bilingual (EN/中文); this makes it trilingual. Three language-capable swarms is three times the simulation surface.

**Social agent tool preservation (PR #232)** — the social agent was silently dropping `tool_calls` when filtering out empty LLM messages. Sims that "ran but agents said nothing" had this as the root cause. Fixed.

**Nemotron data persistence (PR #238)** — Nemotron datasets were getting wiped on `docker compose down`. Fixed with a named volume so backend data survives container recreation.

**Security sweep — three batches (PRs #229, #230, #231)**:
- A+B: safe patch/minor Dependabot bumps across the board
- C: major version gates behind camel-ai MCP — starlette 1.x, cryptography, pyjwt, python-multipart
- D: CVE-2025-71176 — pytest 8→9.1.1 + pytest-asyncio bump

**Community health (PRs #235, #236, #237)**:
- `SECURITY.md` + `CONTRIBUTING.md` + `.github/` consolidation
- `FUNDING.yml` — Sponsor button live on the repo
- LICENSE moved back to root so GitHub license detection works

**Maintenance**: vite 8.1.3 (PR #225), uv group 6-dep bump (PR #233).
