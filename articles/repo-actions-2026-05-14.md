### 5. Private Share Link (HMAC-signed access token)

**Type:** Feature / Security
**Effort:** Small (hours)
**Impact:** Published simulations are world-readable; unpublished simulations are fully gated. There is no middle ground: an operator who wants to share a preliminary result with Lorimer Ventures before publishing — for feedback, review, or an embargoed finding — has no mechanism today. A HMAC-signed, time-limited private share link (`/share/<id>?token=<hex>&expires=<unix>`) grants full share-page access to an unpublished simulation for a configurable window (default 24h, configurable via `PRIVATE_LINK_TTL_HOURS`). `GET /api/simulation/<id>/private-link` (admin-gated) generates a token. The share page honors a valid token identically to a published-sim request, adding only a banner: "🔒 Private link — this simulation is not yet published. Expires [time]." The HMAC signing reuses the same `hmac` + `hashlib.sha256` stdlib pair as PR #79 — zero new deps. The sim remains unpublished; the token expires; the operator retains full control.
**How:**
1. Add `backend/app/services/private_link.py` (~120 LoC, pure stdlib `hmac` + `hashlib` + `os` + `time` + `datetime`). `class PrivateLinkNotConfiguredError(Exception): pass`. `_get_secret() -> bytes`: reads `os.environ.get("PRIVATE_LINK_SECRET", "")` — raises `PrivateLinkNotConfiguredError` if blank. `_get_ttl_seconds() -> int`: reads `PRIVATE_LINK_TTL_HOURS`, parses to int, clamps to `[1, 168]` (1h minimum, 1 week maximum), returns seconds. `generate_token(sim_id: str, expires_at_unix: int) -> str`: `hmac.new(_get_secret(), f"{sim_id}:{expires_at_unix}".encode(), hashlib.sha256).hexdigest()`. `generate_private_link(sim_id, base_url) -> dict`: computes `expires_at = int(time.time()) + _get_ttl_seconds()`, generates token, returns `{url: f"{base_url}/share/{sim_id}?token={token}&expires={expires_at}", expires_at: datetime.utcfromtimestamp(expires_at).isoformat() + "Z", ttl_hours: _get_ttl_seconds() // 3600}`. `verify_token(sim_id, token, expires_str) -> bool`: parses `expires_str` as int (returns `False` on parse error), checks `time.time() < expires_int` (not expired), verifies via `hmac.compare_digest(generate_token(sim_id, expires_int), token)` (constant-time). Returns `False` on any error — never raises. ## Selection Rationale

Five ideas, one batch, one arc: the distribution surface layer is complete (21 zero-dep PRs, #57–#82). This batch goes one layer deeper — integration depth, causal narrative, vector visual, analytic comparison, and institutional pre-publication access.

- **Discord + Slack Rich Notifications** (#1) — RevaultDrops is live. Their Discord server would benefit from formatted MiroShark sim-completion cards. Zero new deps, two new env vars, fire-and-forget delivery. This is the highest-leverage move for the integration tier: it makes RevaultDrops' community feed a live MiroShark distribution channel without the operator writing a single line of integration code.
- **Director Event Timeline** (#2) — Director mode is a differentiating feature with no distribution artifact. The annotated belief chart — "here is the curve; here is where the injection fired" — is the citation surface director mode has needed. All data is on disk; the endpoint is a read-and-format. SVG fallback ensures zero new npm deps regardless of `chartjs-plugin-annotation` availability.
- **Shareable Belief Chart SVG** (#3) — The share card shows the verdict; the notebook shows the analysis. The trajectory chart SVG closes the gap: a vector image of the full belief journey, embeddable in any platform that renders `<img>` tags. Pure stdlib `xml.etree.ElementTree`, same zero-dep approach as the sitemap (PR #82).
- **Comparative Run View** (#4) — The lineage navigator (PR #76) makes related sims navigable. The comparison view makes them analytically comparable. A `/compare?a=<id>&b=<id>` URL is a shareable artifact that shows before/after in one link. Zero new backend code; pure frontend composition of existing `embed-summary` endpoint calls.
- **Private Share Link** (#5) — Lorimer Ventures is watching. An operator who wants to share a preliminary result with a VC before publishing has no mechanism today. A HMAC-signed time-limited token — using the same `hmac` + `hashlib.sha256` stdlib pair as PR #79 — gives institutional observers a configurable access window without making the sim public. Zero new deps.

Each idea is scoped for autonomous implementation: all data on disk or in existing endpoints, pure stdlib, zero new deps, follows the established service/route/EmbedDialog/openapi/test pattern. No ambiguous design decisions.
